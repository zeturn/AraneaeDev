# Code of Conduct

## Our Pledge

We pledge to make participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while staying accountable for impact.
- Keep discussions focused on the project and its users.
- Give and receive feedback with empathy.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Public or private abuse, intimidation, or sustained disruption.
- Publishing private information without explicit permission.
- Any conduct that would reasonably make collaboration unsafe or unwelcome.

## Enforcement

Project maintainers may remove comments, close issues, reject contributions, or limit participation when behavior violates this code of conduct. Serious incidents should be reported privately to the maintainers.

## Scope

This code applies in project spaces, including issues, pull requests, discussions, documentation, and related community channels.
